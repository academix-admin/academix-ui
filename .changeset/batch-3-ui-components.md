Initial release of six more libraries extracted from academix-web:

- `@academix-admin/navigation-bar`: nav/tab bar with normal/float/auto-hide modes.
- `@academix-admin/sidebar`: collapsible sidebar / side navigation.
- `@academix-admin/dialog-viewer`: imperative alert/confirm/destructive dialogs.
- `@academix-admin/side-drawer`: portal-based off-canvas drawer.
- `@academix-admin/bottom-viewer`: imperative bottom-sheet viewer (depends on
  `@academix-admin/modal-sheet`; `motion` peer dependency).
- `@academix-admin/lottie-cache`: framework-agnostic Lottie JSON cache + preloader.
