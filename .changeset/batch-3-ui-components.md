Initial release of six more libraries extracted from academix-web:

- `@academix/navigation-bar`: nav/tab bar with normal/float/auto-hide modes.
- `@academix/sidebar`: collapsible sidebar / side navigation.
- `@academix/dialog-viewer`: imperative alert/confirm/destructive dialogs.
- `@academix/side-drawer`: portal-based off-canvas drawer.
- `@academix/bottom-viewer`: imperative bottom-sheet viewer (depends on
  `@academix/modal-sheet`; `motion` peer dependency).
- `@academix/lottie-cache`: framework-agnostic Lottie JSON cache + preloader.
